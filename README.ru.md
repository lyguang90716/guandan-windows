Guandan Master — проект карточной игры Guandan с упором на локальную игру и поддержкой LAN-мультиплеера.

## Основные возможности

- Полный цикл партии: раздача, ход, пас, смена инициативы, расчет результата, повышение уровня, tribute/return tribute.
- Правила A-уровня: проход, понижение при неудаче, правило "три неудачи — обратно на 2-й уровень".
- Четыре уровня ИИ: `easy / medium / hard / master`.
- Поддержка сборки десктоп-версии через Electron.

## Кратко о правилах

- 4 игрока, 2 команды (напарники сидят напротив).
- Нужно перебивать комбинацией того же типа и большей силы.
- Когда остальные пасуют, последний сыгравший начинает новый раунд.
- Главная цель: дойти до уровня `A` и успешно пройти A-челлендж.

## ИИ-стратегия

- `easy`: базовая корректная игра, дружелюбно для новичков.
- `medium`: кооперация с напарником и избегание невыгодных бомб.
- `hard`: тактический lookahead на 1-2 хода.
- `master`: усиленная эндшпильная логика и модель соперника.

## Быстрый старт

```bash
npm install
npm run dev
```

Опционально для локального сервера:

```bash
node server/index.js
```
